# Results

This directory contains outputs from model training and evaluation.

## Files

- `cv_scores.json` - Cross-validation results with utility scores
- `feature_importance.csv` - Top features ranked by importance
- `performance_comparison.png` - Visual comparison of model performance
- `transaction_cost_analysis.txt` - Analysis of trading costs impact

## Reproducing Results
```bash
python scripts/train_and_evaluate.py
```

Results will be automatically saved to this directory.
